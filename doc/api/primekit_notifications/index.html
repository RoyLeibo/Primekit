notifications library

Primekit Notifications module.

Local notification scheduling, push message handling, slide-in in-app banners, and Android notification channel setup.

import 'package:primekit/notifications.dart';

Classes

InAppBanner
A non-intrusive in-app notification banner that slides in from the top or bottom and auto-dismisses after displayDuration.
InAppBannerConfig
Configuration object passed to InAppBannerService.show.
InAppBannerService
A service for imperatively showing and hiding InAppBanners.
LocalNotifier
Schedules and delivers local notifications via flutter_local_notifications.
NotificationChannel
Describes an Android notification channel.
NotificationTap
Delivered when the user taps a local notification.
PendingNotification
A pending scheduled notification.
PushHandler
Handles FCM / APNs push messages using firebase_messaging.
PushMessage
A normalised push notification message received from FCM / APNs.
