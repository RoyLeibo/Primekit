primekit_permissions

permissions library

Primekit Permissions module.

PermissionGate widget, multi-step permission flow with rationale dialogs, and static permission helper utilities.

import 'package:primekit/permissions.dart';

Classes

Permission
Defines the permissions which can be checked and requested.
PermissionFlow
Displays a multi-step rationale dialog then requests each permission sequentially before returning a PermissionFlowResult.
PermissionFlowResult
The outcome of a completed PermissionFlow.
PermissionGate
A widget that checks and optionally requests a device permission before rendering its child.
PermissionHelper
Static utility methods for querying and requesting device permissions.
PermissionRequest
A single permission request step within a PermissionFlow.

Enums

PermissionStatus
Defines the state of a Permission.
