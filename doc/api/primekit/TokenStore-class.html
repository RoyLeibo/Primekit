TokenStore

TokenStore class final

Secure, singleton storage for JWT access and refresh tokens.

Persists tokens using flutter_secure_storage, which uses the platform Keychain (iOS/macOS) or EncryptedSharedPreferences (Android).

Includes a lightweight JWT expiry check that decodes the exp claim from the token payload without performing cryptographic verification â€” use this only to decide whether to attempt a refresh, not to trust token contents.

final store = TokenStore.instance;
await store.saveAccessToken(accessToken);
final valid = await store.hasValidToken();
