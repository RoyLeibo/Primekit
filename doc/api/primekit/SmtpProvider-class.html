SmtpProvider

SmtpProvider class

Sends emails by posting to a backend relay endpoint that handles SMTP.

Direct SMTP connections from mobile apps are blocked by most carriers and email providers. This provider sends your message payload to a backend endpoint you control, which then relays it via SMTP.

Backend contract: POST to relayUrl with JSON body:

{
  "to": "...",
  "subject": "...",
  "text": "...",
  "html": "...",
  "replyTo": "...",
  "headers": {}
}

The backend must return { "messageId": "..." } on success.

For a simple backend relay using Node.js + Nodemailer, see the Primekit documentation.
