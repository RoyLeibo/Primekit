SyncRepository

SyncRepository<T> class

Generic offline-first repository that transparently persists data locally and synchronises with a remote SyncDataSource.

All write operations (create, update, delete) return immediately after writing to the local cache and enqueueing a SyncChange. Background sync is triggered periodically via syncInterval and can also be triggered manually via syncNow.

Conflict resolution is handled by a pluggable ConflictResolver; the default is LastWriteWinsResolver.

final repo = SyncRepository<Todo>(
  collection: 'todos',
  remoteSource: FirestoreSyncSource(),
  fromJson: Todo.fromJson,
  conflictResolver: LastWriteWinsResolver(),
  syncInterval: const Duration(minutes: 5),
);

// Listen for reactive updates
repo.watchAll().listen((todos) => setState(() => _todos = todos));

// Write (offline-safe)
await repo.create({'title': 'Buy milk', 'done': false});
await repo.syncNow();
