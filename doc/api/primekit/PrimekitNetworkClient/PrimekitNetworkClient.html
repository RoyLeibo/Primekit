Creates a PrimekitNetworkClient.

baseUrl is required. headers are merged with the default headers (Accept: application/json, Content-Type: application/json). timeout applies to both connection and receive operations (default 30s). onAuthToken is an optional async callback that returns the current bearer token; it is called before each request.
