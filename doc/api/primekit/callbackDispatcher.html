callbackDispatcher function

void callbackDispatcher()

Top-level callback registered with WorkManager.

Must be a free function (not a method) so it can be called on the WorkManager background isolate.

Implementation

@pragma('vm:entry-point')
void callbackDispatcher() {
  wm.Workmanager().executeTask(
    (String taskName, Map<String, dynamic>? inputData) =>
        TaskScheduler._dispatchTask(
      taskName: taskName,
      inputData: inputData ?? {},
    ),
  );
}
