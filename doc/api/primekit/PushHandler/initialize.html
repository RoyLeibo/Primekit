Initializes FCM message handling.

onMessage is called when a push arrives while the app is in the foreground.

onMessageOpenedApp is called when the user taps a notification that was displayed while the app was in the background or terminated.

onTokenRefresh is called when the FCM token is created or rotated. Use this to send the updated token to your backend.

When firebase_messaging is installed, replace the body of this method with the real SDK integration shown in the inline comment.
