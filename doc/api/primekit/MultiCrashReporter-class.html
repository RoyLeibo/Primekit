MultiCrashReporter

MultiCrashReporter class

A CrashReporter that fans out every call to multiple reporters.

Use this when you want to send crash data to more than one backend simultaneously (e.g. both Firebase Crashlytics and Sentry):

final reporter = MultiCrashReporter([
  FirebaseCrashReporter(),
  SentryCrashReporter(dsn: 'https://xxx@sentry.io/123'),
]);
await reporter.initialize();
