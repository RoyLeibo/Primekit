AnalyticsEvent

AnalyticsEvent class final

Represents a single analytics event with a name, optional parameters, and a timestamp.

Use the named factory constructors for well-known event types, or supply a custom name directly:

// Predefined factory
final event = AnalyticsEvent.screenView(screenName: 'HomeScreen');

// Custom event
final event = AnalyticsEvent(
  name: 'level_complete',
  parameters: {'level': 3, 'score': 9500},
);
