ValidationResult

ValidationResult class final

The result of a schema validation operation.

Use ValidationResult.valid when all rules pass and ValidationResult.invalid when one or more rules fail.

final result = schema.validate(input);
if (result.isValid) {
  processData(result.value);
} else {
  showErrors(result.errors);
}
