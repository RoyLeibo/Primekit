OtpService

OtpService class final

Generates and validates time-limited one-time passwords (OTPs) for email/SMS authentication flows.

OTPs are stored in-memory only â€” they are not persisted across restarts, which is intentional: OTPs should be short-lived and re-sent on app restart. Use server-side storage when cross-process persistence is required.

// Generate & store
final code = service.generate();
service.store('user@example.com', code, ttl: Duration(minutes: 5));

// Validate on user input
final result = service.validate('user@example.com', userInput);
