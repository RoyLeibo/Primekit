LazyList

LazyList<T> class

An infinite-scroll list that automatically loads pages as the user approaches the bottom.

onLoadPage is called with the current zero-based page index and pageSize. When it returns an empty list the list treats that as the last page and stops requesting more data.

LazyList<User>(
  onLoadPage: (page, pageSize) => api.getUsers(page: page, limit: pageSize),
  itemBuilder: (context, user, index) => UserTile(user: user),
)
