AppPreferences

AppPreferences class final

A typed SharedPreferences wrapper for common application settings.

Covers the three most common preferences out of the box — theme mode, locale, and onboarding completion — and provides a generic set/get pair for arbitrary typed values (String, bool, int, double, and JSON maps).

final prefs = AppPreferences.instance;
await prefs.setThemeMode(ThemeMode.dark);
final mode = await prefs.getThemeMode(); // ThemeMode.dark
