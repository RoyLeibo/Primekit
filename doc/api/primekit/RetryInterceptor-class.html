RetryInterceptor

RetryInterceptor class final

A Dio Interceptor that automatically retries failed requests with exponential backoff.

Retry behaviour:

  • Retries on network errors (no response received) and 5xx server errors.
  • Does not retry on 4xx client errors â€” these indicate a problem with the request itself that a retry cannot fix.
  • Each retry waits initialDelay * 2^(attempt - 1) before executing, capped at maxDelay.
  • After maxRetries failed attempts the original error is rethrown.
final dio = Dio();
dio.interceptors.add(
  RetryInterceptor(
    maxRetries: 3,
    initialDelay: const Duration(seconds: 1),
  ),
);
