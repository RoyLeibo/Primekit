CachedFlagProvider

CachedFlagProvider class final

A caching decorator for any FlagProvider.

Wraps a delegate provider and adds:

  • In-memory + SharedPreferences persistence across app restarts.
  • TTL-based expiry — values are considered fresh for ttl after the last fetch.
  • Stale-while-revalidate — stale values are returned immediately while a background refresh is triggered concurrently.
final cached = CachedFlagProvider(
  delegate: FirebaseFlagProvider(),
  ttl: const Duration(hours: 2),
);
FlagService.instance.configure(cached);
