InAppBanner

InAppBanner class

A non-intrusive in-app notification banner that slides in from the top or bottom and auto-dismisses after displayDuration.

Supports tap, swipe-to-dismiss, and programmatic dismissal via InAppBannerService.hide.

Typically you use InAppBannerService.show rather than placing this widget directly in the tree.
