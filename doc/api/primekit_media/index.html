primekit_media

media library

Primekit Media module.

Image picker, compression, crop, and upload pipeline with progress stream. Pluggable upload backends (Firebase Storage, custom).

import 'package:primekit/media.dart';

Wraps ImageCropper with a clean immutable-result API.
MediaFile
An immutable value type representing a picked or compressed media file.
MediaPicker
